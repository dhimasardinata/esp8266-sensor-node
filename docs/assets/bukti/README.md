## Tips pengambilan bukti

- Pastikan timestamp perangkat/laptop terlihat.
- Untuk serial monitor, capture bagian yang memuat kata kunci hasil uji.
- Untuk dashboard, tampilkan nilai sensor + waktu pengambilan data.
- Untuk uji akurasi, foto sensor node dan alat validator dalam 1 frame.
- Simpan dalam resolusi cukup (minimal 1280 px lebar) agar terbaca saat dicetak.
